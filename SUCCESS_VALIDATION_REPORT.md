### ✅ Query 5: 什么是市盈率？

**路由**: KNOWLEDGE
**LLM生成**: ✅ 成功
**质量评分**: ⭐⭐⭐⭐⭐

**AI分析**:
```markdown
# 📖 定义
市盈率（Price-to-Earnings Ratio，简称PE）是股票价格与每股收益的比率。它是衡量股票估值水平最常用、最核心的指标之一，其本质反映了投资者为了获得公司**1元**的净利润，愿意支付多少元的股价。

# 📐 计算方法/核心区别
市盈率的计算公式为：
**市盈率（PE）= 股价 / 每股收益（EPS）**

在实际应用中，根据所采用的每股收益数据不同，市盈率主要分为两种：
*   **静态市盈率**：使用公司过去**12个月**（即最近一个完整财年）的每股收益进行计算。它反映的是基于历史业绩的估值。
*   **动态市盈率**：使用市场对公司**未来12个月**的预测每股收益进行计算。它更侧重于反映公司未来的成长预期。

# 🔍 实际应用
投资者通常使用市盈率来初步判断股票的估值高低，并进行横向或纵向比较。

1.  **横向比较（同行业）**：例如，投资者在比较两家同行业的科技公司A和B时，发现A公司市盈率为**20倍**，而B公司为**35倍**。从数据来看，市场为B公司的每元利润支付了更高的价格，这可能意味着市场更看好B公司的未来增长潜力，或者A公司的估值相对更具吸引力。但必须结合公司具体的增长速度和盈利能力进一步分析。

2.  **纵向比较（自身历史）**：投资者可以观察一家公司自身市盈率的历史波动范围。如果其当前市盈率显著低于历史平均水平，可能意味着股价处于相对低估的区间；反之，则可能意味着估值偏高。但这需要结合公司当前所处的商业周期和行业环境来判断。

# ⚠️ 注意事项
使用市盈率时需注意以下几点限制：
*   **行业差异巨大**：不同行业的合理市盈率范围天差地别。例如，高成长性的科技行业市盈率通常较高（如**25倍**以上），而增长稳定的公用事业或银行业市盈率则普遍较低（可能仅在**10倍**左右）。因此，**"PE<15为低估，>25为高估"** 只是一个非常粗略的参考，绝不能跨行业套用。
*   **不适用于亏损或微利公司**：当公司净利润为负或极低时，市盈率会失去意义或变得异常高，此时需要参考市销率（PS）、市净率（PB）等其他估值指标。
*   **盈利质量的影响**：市盈率计算中的"收益"是会计净利润，可能受到一次性损益、折旧政策等影响，未必能真实反映公司持续的盈利能力。分析时需考察盈利的构成与质量。
```
