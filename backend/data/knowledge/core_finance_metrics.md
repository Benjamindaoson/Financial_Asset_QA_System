# 核心金融指标 (Core Finance Metrics)

## 估值指标

### 市盈率 (P/E Ratio)

**定义**：市盈率是股票价格与每股收益的比率，用于衡量股票估值水平。

**公式**：
```
市盈率 (P/E) = 股价 / 每股收益 (EPS)
```

**解读**：
- P/E < 15：可能被低估
- P/E 15-25：合理估值区间
- P/E > 25：可能被高估（成长股除外）

**注意事项**：
- 不同行业的合理P/E区间不同
- 周期性行业在景气高点P/E反而低
- 亏损公司P/E无意义

---

### 市净率 (P/B Ratio)

**定义**：市净率是股价与每股净资产的比率。

**公式**：
```
市净率 (P/B) = 股价 / 每股净资产
```

**解读**：
- P/B < 1：股价低于净资产，可能被低估
- P/B 1-3：正常估值
- P/B > 3：高估值，需关注ROE是否匹配

**适用场景**：
- 银行、地产等重资产行业
- 不适用于轻资产科技公司

---

### 每股收益 (EPS)

**定义**：公司净利润除以总股本。

**公式**：
```
EPS = 净利润 / 总股本
```

**类型**：
- **基本EPS**：不考虑稀释
- **稀释EPS**：考虑期权、可转债等潜在稀释

---

### PEG比率

**定义**：市盈率相对盈利增长比率。

**公式**：
```
PEG = P/E / 净利润增长率(%)
```

**解读**：
- PEG < 1：估值合理或被低估
- PEG = 1：估值与增长匹配
- PEG > 1：估值偏高

**适用场景**：成长股估值

---

### 净资产收益率 (ROE)

**定义**：衡量股东权益回报率。

**公式**：
```
ROE = 净利润 / 股东权益 × 100%
```

**杜邦分析**：
```
ROE = 净利率 × 资产周转率 × 权益乘数
```

**解读**：
- ROE > 15%：优秀
- ROE 10-15%：良好
- ROE < 10%：一般

---

### EV/EBITDA

**定义**：企业价值倍数，考虑债务的估值指标。

**公式**：
```
EV = 市值 + 净债务
EBITDA = 息税折旧摊销前利润
EV/EBITDA = 企业价值 / EBITDA
```

**优势**：
- 消除资本结构差异
- 适合跨国比较
- 适合并购估值

---

## 盈利能力指标

### 毛利率

**公式**：
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```

### 净利率

**公式**：
```
净利率 = 净利润 / 营业收入 × 100%
```

---

## 偿债能力指标

### 资产负债率

**公式**：
```
资产负债率 = 总负债 / 总资产 × 100%
```

**安全线**：
- < 50%：安全
- 50-70%：正常
- > 70%：高风险

### 流动比率

**公式**：
```
流动比率 = 流动资产 / 流动负债
```

**标准**：
- > 2：偿债能力强
- 1-2：正常
- < 1：短期偿债压力大

---

## 现金流指标

### 自由现金流 (FCF)

**公式**：
```
FCF = 经营活动现金流 - 资本支出
```

**意义**：真正可供股东分配的现金

### 现金流量比率

**公式**：
```
现金流量比率 = 经营活动现金流 / 流动负债
```

---

## 成长性指标

### 营收增长率

**公式**：
```
营收增长率 = (本期营收 - 上期营收) / 上期营收 × 100%
```

### 净利润增长率

**公式**：
```
净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润 × 100%
```

---

## 使用建议

1. **综合分析**：不要单看一个指标
2. **行业对比**：与同行业公司比较
3. **历史趋势**：关注指标变化趋势
4. **质量优先**：ROE、现金流比利润更重要
