# 股票基础知识

## 股票市场分类

### A股市场
- **交易所**：上海证券交易所（SSE）、深圳证券交易所（SZSE）
- **交易时间**：周一至周五 9:30-11:30, 13:00-15:00
- **交易制度**：T+1（当天买入，次日才能卖出）
- **涨跌幅限制**：
  - 主板：±10%
  - 科创板/创业板：±20%
  - ST股票：±5%
  - 新股上市首日：无涨跌幅限制（科创板/创业板前5日）

### 港股市场
- **交易所**：香港交易所（HKEX）
- **交易时间**：
  - 早市：9:30-12:00
  - 午市：13:00-16:00
- **交易制度**：T+0（当天可买卖）
- **涨跌幅限制**：无涨跌幅限制
- **最小变动单位**：1手（不同股票手数不同，100股、500股等）

### 美股市场
- **主要交易所**：纽约证券交易所（NYSE）、纳斯达克（NASDAQ）
- **交易时间**（美东时间）：
  - 常规交易：9:30-16:00
  - 盘前交易：4:00-9:30
  - 盘后交易：16:00-20:00
- **交易制度**：T+0（无限制日内交易）
- **涨跌幅限制**：无涨跌幅限制
- **熔断机制**：
  - 下跌7%：暂停15分钟
  - 下跌13%：暂停15分钟
  - 下跌20%：全天停止交易

## 交易规则

### 竞价方式

**集合竞价**（A股）：
- 9:15-9:25：开盘集合竞价
- 14:57-15:00：收盘集合竞价
- 按价格优先、时间优先原则撮合

**连续竞价**：
- 正常交易时段
- 实时撮合成交

### 委托类型

1. **市价委托**：按当前市场价格立即成交
2. **限价委托**：指定价格，达到价格才成交
3. **止损委托**：价格跌破设定值自动卖出
4. **止盈委托**：价格涨到设定值自动卖出

## 融资融券

### 融资（买入）
- **定义**：向券商借钱买股票
- **保证金比例**：通常50%（即借1万需自有5千）
- **利率**：年化8-10%
- **风险**：放大收益也放大亏损

### 融券（卖空）
- **定义**：向券商借股票卖出，未来买回归还
- **适用场景**：看跌时做空获利
- **保证金比例**：通常50%
- **费用**：融券费率 + 利息

### 强制平仓
- **维持担保比例** < 130%时，券商有权强制平仓
- 计算公式：
```
维持担保比例 = (现金 + 股票市值) / (融资负债 + 融券负债) × 100%
```

## 股票质押

### 定义
股东将持有的股票质押给金融机构，获得资金。

### 质押率
- 通常为股票市值的30-50%
- 蓝筹股质押率较高
- 小盘股质押率较低

### 风险
- 股价下跌触及平仓线，需补仓或被强制平仓
- 大股东质押比例过高（>50%）是风险信号

## 股票分类

### 按市值分类
- **大盘股**：市值 > 500亿
- **中盘股**：市值 50-500亿
- **小盘股**：市值 < 50亿

### 按行业分类
- **周期股**：钢铁、煤炭、化工（随经济周期波动）
- **消费股**：食品饮料、医药（业绩稳定）
- **成长股**：科技、新能源（高增长高估值）
- **金融股**：银行、保险、券商

### 按概念分类
- **蓝筹股**：业绩优良、分红稳定的大公司
- **白马股**：业绩优秀、信息透明
- **ST股**：连续两年亏损，有退市风险
- **次新股**：上市1年内的股票

## 交易费用

### A股
- **佣金**：0.02%-0.03%（双向）
- **印花税**：0.1%（仅卖出）
- **过户费**：0.001%（双向）

### 港股
- **佣金**：0.25%左右
- **印花税**：0.1%（双向）
- **交易征费**：0.0027%
- **交易费**：0.005%

### 美股
- **佣金**：多数券商已免佣
- **SEC费用**：卖出时收取少量费用

## 风险提示

1. **市场风险**：股价波动可能导致本金损失
2. **流动性风险**：小盘股可能无法及时卖出
3. **政策风险**：监管政策变化影响股价
4. **公司风险**：业绩下滑、财务造假、退市风险
5. **杠杆风险**：融资融券放大亏损
