# 宏观经济指标

## GDP (国内生产总值)

### 定义
GDP（Gross Domestic Product）是一个国家或地区在一定时期内生产的所有最终商品和服务的市场价值总和，是衡量经济规模和增长的核心指标。

### 计算方法

#### 生产法
```
GDP = 各行业增加值之和
增加值 = 总产出 - 中间投入
```

#### 支出法
```
GDP = 消费(C) + 投资(I) + 政府支出(G) + 净出口(X-M)
```

#### 收入法
```
GDP = 劳动者报酬 + 生产税净额 + 固定资产折旧 + 营业盈余
```

### 关键指标

#### GDP增长率
```
实际GDP增长率 = (本期实际GDP - 上期实际GDP) / 上期实际GDP × 100%
```

**判断标准**：
- 高速增长：>8%
- 中速增长：6-8%
- 中低速增长：4-6%
- 低速增长：<4%

#### 人均GDP
```
人均GDP = GDP总量 / 总人口
```
反映平均生活水平和经济发展程度

### 对市场影响
- **GDP超预期**：经济向好，股市利好，货币可能升值
- **GDP低于预期**：经济疲软，可能出台刺激政策
- **GDP增速放缓**：央行可能降息降准，债市利好

## CPI (消费者物价指数)

### 定义
CPI（Consumer Price Index）衡量一篮子消费品和服务价格变化，是通货膨胀的主要指标。

### 构成
- **食品烟酒**：权重约30%
- **居住**：权重约20%
- **交通通信**：权重约15%
- **教育文化娱乐**：权重约10%
- **医疗保健**：权重约8%
- **衣着、生活用品等**：其余权重

### 计算方法
```
CPI = (本期一篮子商品价格 / 基期一篮子商品价格) × 100
CPI同比 = (本月CPI - 去年同月CPI) / 去年同月CPI × 100%
```

### 通胀判断
- **CPI > 3%**：通胀压力，央行可能加息
- **2% < CPI < 3%**：温和通胀，经济健康
- **1% < CPI < 2%**：低通胀
- **CPI < 1%**：通缩风险

### 对市场影响
- **高通胀**：央行加息预期，债市下跌，股市承压
- **低通胀/通缩**：宽松货币政策，债市利好
- **结构性通胀**：关注受益行业（如食品价格上涨利好农业）

## 利率政策

### 基准利率

#### 存款准备金率
商业银行存放在央行的准备金占存款总额的比例。

**调整影响**：
- **降准**：释放流动性，利好股市债市
- **加准**：收紧流动性，市场承压

#### 贷款市场报价利率 (LPR)
银行对优质客户的贷款利率，是贷款定价基准。

**期限结构**：
- 1年期LPR：短期贷款基准
- 5年期LPR：房贷基准

**调整影响**：
- **LPR下调**：降低融资成本，利好实体经济和股市
- **LPR上调**：抑制过热，地产等高杠杆行业承压

### 公开市场操作

#### 逆回购
央行向市场投放短期流动性的工具。

**操作方式**：
- 7天逆回购：最常用
- 14天、28天逆回购：跨节日流动性

#### 中期借贷便利 (MLF)
央行向商业银行提供中期资金的工具，期限通常1年。

**利率传导**：
MLF利率 → LPR → 贷款利率

### 利率对资产影响

**降息周期**：
- 股市：估值提升，成长股受益
- 债市：债券价格上涨
- 房地产：融资成本降低，销售改善
- 黄金：无息资产吸引力增强

**加息周期**：
- 股市：估值承压，价值股相对抗跌
- 债市：债券价格下跌
- 房地产：融资成本上升，需求下降
- 美元：加息货币升值

## 汇率

### 定义
汇率是两种货币之间的兑换比率，反映货币的相对价值。

### 汇率标价

#### 直接标价法
```
1美元 = 7.2人民币
```
外币作为基准，本币表示价格

#### 间接标价法
```
1人民币 = 0.139美元
```
本币作为基准，外币表示价格

### 汇率影响因素

#### 经济基本面
- GDP增速：经济强劲货币升值
- 通胀水平：高通胀货币贬值
- 贸易顺差：出口大于进口，货币升值

#### 利率差异
- 利率高的货币吸引资本流入，升值
- 利差扩大，套利交易增加

#### 政策干预
- 央行买入本币：货币升值
- 央行卖出本币：货币贬值

### 汇率对市场影响

#### 人民币升值
- **出口企业**：竞争力下降，利润承压
- **进口企业**：成本降低，利润改善
- **航空公司**：美元债务减少，利好
- **海外收入企业**：折算收入减少

#### 人民币贬值
- **出口企业**：竞争力提升，订单增加
- **进口企业**：成本上升
- **外债企业**：偿债压力增加
- **资本外流**：股市承压

### 汇率与大类资产

**美元指数上涨**：
- 新兴市场货币贬值
- 大宗商品价格下跌
- 黄金价格承压
- 美股相对强势

**美元指数下跌**：
- 新兴市场货币升值
- 大宗商品价格上涨
- 黄金价格上涨
- 国际资本流向新兴市场
