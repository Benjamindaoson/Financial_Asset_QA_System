# 估值指标

## 市盈率 (PE Ratio)

### 定义
市盈率（Price-to-Earnings Ratio）是股票价格与每股收益的比率，是衡量股票估值水平最常用的指标之一。

### 计算方法
```
市盈率 = 股票价格 / 每股收益(EPS)
```

其中，每股收益可以使用：
- 静态PE：使用上一年度的实际EPS
- 动态PE：使用当前财年的预测EPS
- 滚动PE(TTM)：使用最近12个月的实际EPS

### 应用场景
1. **横向比较**：比较同行业不同公司的估值水平，PE较低可能意味着被低估
2. **纵向比较**：观察同一公司历史PE区间，判断当前估值位置
3. **市场情绪**：整体市场PE水平反映投资者风险偏好
4. **投资决策**：结合成长性，PEG指标(PE/增长率)更全面

### 注意事项
- 周期性行业在盈利高点PE反而低，需谨慎使用
- 亏损企业PE为负，不适用此指标
- 不同行业PE中枢差异大，需行业内比较

## 市净率 (PB Ratio)

### 定义
市净率（Price-to-Book Ratio）是股票价格与每股净资产的比率，反映市场对公司净资产的溢价程度。

### 计算方法
```
市净率 = 股票价格 / 每股净资产(BVPS)
每股净资产 = 股东权益 / 总股本
```

### 应用场景
1. **资产密集型行业**：银行、地产等重资产行业更适合用PB估值
2. **破产清算价值**：PB<1可能意味着股价低于清算价值
3. **ROE结合分析**：高ROE企业应享有更高PB溢价
4. **周期底部判断**：周期股在底部PB往往接近历史低位

## 市销率 (PS Ratio)

### 定义
市销率（Price-to-Sales Ratio）是公司市值与营业收入的比率，适用于尚未盈利但有收入的成长型企业。

### 计算方法
```
市销率 = 总市值 / 营业收入
或 PS = 股价 / 每股营业收入
```

### 应用场景
1. **亏损企业估值**：互联网、生物医药等前期亏损企业
2. **收入质量评估**：结合净利率分析盈利能力
3. **成长性判断**：高增长企业可承受较高PS
4. **行业比较**：SaaS、电商等轻资产行业常用

## EV/EBITDA

### 定义
企业价值倍数，用企业价值(EV)除以息税折旧摊销前利润(EBITDA)，剔除资本结构和会计政策影响。

### 计算方法
```
EV = 市值 + 净债务（总债务 - 现金）
EBITDA = 净利润 + 所得税 + 利息 + 折旧 + 摊销
EV/EBITDA = 企业价值 / EBITDA
```

### 应用场景
1. **并购估值**：收购方关注企业整体价值而非股权价值
2. **跨国比较**：剔除税率差异影响
3. **资本密集型行业**：折旧摊销大的制造业、基建等
4. **杠杆比较**：不同负债率企业的可比估值

### 优势
- 不受资本结构影响，高负债和低负债企业可比
- 不受折旧政策影响，更接近现金流
- 适合并购场景的企业整体估值
