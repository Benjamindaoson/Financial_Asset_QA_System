# 财务报表分析

## 利润表 (Income Statement)

### 核心指标

#### 营业收入
企业主营业务和其他业务的收入总和，是利润的源头。

**分析要点**：
- 收入增长率：同比、环比增速反映业务扩张能力
- 收入结构：主营占比、业务线贡献分析
- 收入质量：关联交易、预收账款变化

#### 净利润
企业最终盈利水平，扣除所有成本费用和税费后的利润。

**计算路径**：
```
营业收入
- 营业成本
- 税金及附加
= 营业利润
+ 营业外收支
= 利润总额
- 所得税
= 净利润
```

**关键比率**：
- 毛利率 = (收入-成本)/收入：反映产品竞争力
- 净利率 = 净利润/收入：综合盈利能力
- ROE = 净利润/股东权益：股东回报率

### 利润质量分析

1. **扣非净利润**：扣除非经常性损益后的净利润，更能反映主业盈利
2. **经营性利润占比**：营业利润/利润总额，避免依赖投资收益或政府补助
3. **利润含金量**：经营现金流/净利润，>1说明利润能转化为现金

## 资产负债表 (Balance Sheet)

### 资产结构

#### 流动资产
- **货币资金**：现金及现金等价物，流动性最强
- **应收账款**：赊销形成，需关注账龄和坏账准备
- **存货**：原材料、在产品、产成品，周转率反映运营效率

#### 非流动资产
- **固定资产**：厂房设备等，资本密集度指标
- **无形资产**：专利、商誉等，关注减值风险
- **长期股权投资**：对外投资，关注投资收益

### 负债与权益

#### 负债分析
```
资产负债率 = 总负债/总资产
流动比率 = 流动资产/流动负债
速动比率 = (流动资产-存货)/流动负债
```

**健康标准**：
- 资产负债率：制造业50-70%，轻资产30-50%
- 流动比率：>1.5较安全
- 速动比率：>1较安全

#### 股东权益
- 实收资本：股东投入
- 资本公积：溢价发行等
- 盈余公积：利润留存
- 未分配利润：可分配利润

## 现金流量表 (Cash Flow Statement)

### 三大现金流

#### 经营活动现金流
企业日常经营产生的现金流入流出，是最重要的现金流。

**核心指标**：
```
经营现金流净额 = 销售商品收到的现金 - 购买商品支付的现金 - 支付职工薪酬等
```

**质量判断**：
- 经营现金流>净利润：利润含金量高
- 经营现金流持续为正：造血能力强
- 收现比 = 销售收现/营业收入：>1说明回款良好

#### 投资活动现金流
对外投资、购建固定资产等产生的现金流。

**分析要点**：
- 资本支出(CAPEX)：购建固定资产支出，反映扩张力度
- 自由现金流 = 经营现金流 - 资本支出：可自由支配现金
- 投资收益现金：处置资产、收到股利等

#### 筹资活动现金流
融资、还债、分红等产生的现金流。

**关注点**：
- 借款增减：融资需求和能力
- 股利支付：分红政策
- 股权融资：增发、回购

### 现金流健康度评估

**优秀企业特征**：
- 经营现金流持续为正且增长
- 自由现金流充裕
- 现金流覆盖负债能力强

**预警信号**：
- 经营现金流为负但净利润为正（利润不能变现）
- 依赖筹资现金流维持运营
- 现金流持续恶化
