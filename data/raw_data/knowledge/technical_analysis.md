# 技术分析指标

## 移动平均线 (Moving Average)

### 定义
移动平均线是将一定时期内的价格平均值连成曲线，用于平滑价格波动，识别趋势方向。

### 计算方法

#### 简单移动平均线 (SMA)
```
SMA(n) = (P1 + P2 + ... + Pn) / n
```
其中P为收盘价，n为周期数

#### 指数移动平均线 (EMA)
```
EMA(today) = Price(today) × K + EMA(yesterday) × (1-K)
K = 2 / (n + 1)
```
EMA对近期价格赋予更高权重，反应更灵敏

### 常用周期
- **短期均线**：5日、10日，反映短期趋势
- **中期均线**：20日、30日、60日，反映中期趋势
- **长期均线**：120日、250日，反映长期趋势

### 应用场景

#### 趋势判断
- 价格在均线上方：上升趋势
- 价格在均线下方：下降趋势
- 均线向上：多头市场
- 均线向下：空头市场

#### 金叉与死叉
- **金叉**：短期均线上穿长期均线，买入信号
- **死叉**：短期均线下穿长期均线，卖出信号
- 经典组合：5日线与10日线、20日线与60日线

#### 支撑与压力
- 上升趋势中均线提供支撑
- 下降趋势中均线形成压力
- 重要均线（如60日、250日）支撑压力作用更强

## MACD指标

### 定义
MACD（Moving Average Convergence Divergence）是趋势跟踪型动量指标，通过快慢均线的差值判断趋势和买卖时机。

### 计算方法
```
DIF = EMA(12) - EMA(26)
DEA = EMA(DIF, 9)
MACD柱 = (DIF - DEA) × 2
```

### 三线含义
- **DIF线（快线）**：短期与长期均线的差值
- **DEA线（慢线）**：DIF的9日均线，信号线
- **MACD柱**：DIF与DEA的差值，动能强度

### 应用场景

#### 金叉死叉
- **金叉**：DIF上穿DEA，买入信号
  - 零轴上方金叉：强势买入
  - 零轴下方金叉：弱势反弹
- **死叉**：DIF下穿DEA，卖出信号
  - 零轴上方死叉：强势回调
  - 零轴下方死叉：弱势下跌

#### 背离分析
- **顶背离**：价格创新高，MACD不创新高，见顶信号
- **底背离**：价格创新低，MACD不创新低，见底信号
- 背离是趋势反转的重要预警

#### 零轴判断
- MACD在零轴上方：多头市场
- MACD在零轴下方：空头市场
- 零轴附近震荡：趋势不明

## RSI相对强弱指标

### 定义
RSI（Relative Strength Index）衡量价格涨跌动能，判断超买超卖状态。

### 计算方法
```
RS = n日内上涨幅度平均值 / n日内下跌幅度平均值
RSI = 100 - (100 / (1 + RS))
```
常用周期：6日、12日、24日

### 数值含义
- **RSI > 80**：超买区，可能回调
- **RSI > 70**：强势区，警惕风险
- **30 < RSI < 70**：正常波动区间
- **RSI < 30**：超卖区，可能反弹
- **RSI < 20**：严重超卖，关注机会

### 应用场景

#### 超买超卖
- RSI进入超买区后回落，卖出信号
- RSI进入超卖区后回升，买入信号
- 强势市场中超买可持续，需结合趋势

#### 背离信号
- **顶背离**：价格新高，RSI不创新高，看跌
- **底背离**：价格新低，RSI不创新低，看涨
- RSI背离比价格背离更可靠

#### 中轴线
- RSI突破50：由弱转强
- RSI跌破50：由强转弱
- 50线是多空分界线

## 布林带 (Bollinger Bands)

### 定义
布林带由中轨（移动平均线）和上下轨（标准差通道）组成，衡量价格波动范围和超买超卖。

### 计算方法
```
中轨 = n日移动平均线
上轨 = 中轨 + k × 标准差
下轨 = 中轨 - k × 标准差
```
常用参数：n=20，k=2

### 应用场景

#### 通道突破
- 价格突破上轨：强势信号，可能继续上涨
- 价格跌破下轨：弱势信号，可能继续下跌
- 突破后回归中轨：正常回调

#### 收缩与扩张
- **收缩**：波动率降低，酝酿突破
- **扩张**：波动率增加，趋势启动
- 收缩后扩张往往伴随大行情

#### 支撑压力
- 上轨提供压力位
- 下轨提供支撑位
- 中轨是重要的多空分界线
