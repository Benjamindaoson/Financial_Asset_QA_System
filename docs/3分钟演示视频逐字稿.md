# 金融资产智能问答系统 · 3 分钟演示视频逐字稿

> **目标受众**：CTO、CEO  
> **时长**：约 3 分钟（约 480 字，语速 160 字/分）  
> **核心策略**：用「问题 → 方案 → 证据」三段式，让技术决策者看到架构深度，让业务决策者看到商业价值

---

## 【0:00 - 0:25】开场：痛点与定位

各位好。今天演示的是「华尔街见闻」金融资产智能问答系统。

金融 AI 最大的挑战不是「能不能答」，而是「敢不敢信」。  
数值一旦编造，来源一旦不可追溯，在机构场景就是不可接受的风险。

我们的核心设计理念只有一句话：**系统为骨，数据为血，AI 为脑**。  
把大模型从「全能的生成器」重塑为「克制的分析引擎」——  
**数值必走 API，结论必可溯源，投资建议一律拒答。**

---

## 【0:25 - 0:55】系统整体介绍

系统支持四类问题：**行情、概念、新闻、混合**。

- **行情类**：实时价格、涨跌幅、历史走势、技术指标——全部来自 yfinance、akshare、Stooq 等真实 API，有三级 fallback，保证「有数据才答」。
- **概念类**：市盈率、夏普比率、MACD 原理——走 RAG 知识库，11 个专业文档、向量 + BM25 混合检索，回答有出处。
- **混合类**：比如「苹果最近为什么涨」——同时拉行情、新闻、财报、知识库，多源交叉印证，再生成分析。

前端采用 SSE 流式输出：**图表秒现，文字渐入**。  
数据卡片先渲染，用户不用等 LLM 打完字才看到结果。

---

## 【0:55 - 1:25】资产问答示例

我们现场演示两个典型问题。

**第一个：行情类。**  
输入：「苹果公司最新股价是多少？」  
系统会：提取 ticker AAPL → 调用 get_price → 返回实时价格、涨跌幅、成交量。  
如果用户问「过去一个月走势」，还会自动计算 RSI、MACD、布林带、支撑压力位，并以图表和表格呈现。

**第二个：混合类。**  
输入：「阿里巴巴最近为何大涨？」  
系统会并发执行：  
① get_change 验证涨跌幅；  
② search_web 拉取 Bloomberg、Reuters 等白名单新闻；  
③ search_sec 查 SEC 最新 10-K/10-Q；  
④ search_knowledge 补充相关概念。  
最后把多源数据统一校验后，生成客观分析——**不预测、不推荐，只陈述事实和已有数据。**

---

## 【1:25 - 1:55】RAG 示例

RAG 是知识类问题的核心。我们采用 **向量 + BM25 + RRF 融合**，而不是单一向量检索。

为什么？  
纯向量容易漏掉精确匹配，比如「PE 公式」；纯 BM25 对同义表达不敏感。  
RRF 融合两者，兼顾语义和关键词，金融定义类问题的召回率显著提升。

检索链路：BGE-base-zh 向量化 → BM25 分词 → RRF 融合排序 → BGE-reranker 精排 → Top-K 注入 LLM。  
知识库覆盖：股票基础、估值指标、技术分析、期权期货、风险管理等 11 个专业文档，支持 HyDE、Multi-Query 等查询改写，进一步提升召回。

用户问「什么是夏普比率」，系统会从《风险管理》文档中精准召回定义和公式，并标注来源——**可追溯、可审计。**

## 【1:55 - 2:40】架构说明

架构上，我们**拒绝黑盒 Agent 模式**，采用**确定性多阶段流水线**。

五个显式阶段：**接入 → 编排 → 执行 → 校验 → 合成**。  
每一步都可独立审计、可降级。

- **双路由**：规则路由保证行情类 100% 走 API，不依赖 LLM 判断；LLM 路由处理模糊意图。  
- **数据与排版分离**：客观数据由后端编排为 chart、table、bullets 等原生 blocks，LLM 只负责写文字结论，不参与数值计算——从源头杜绝数值幻觉。  
- **双重反幻觉**：DataValidator 校验「有数可答」；ResponseGuard 校验「LLM 输出的数值是否均来自工具 payload」——输入输出两端封堵。  
- **合规前置**：含「买入」「推荐」「目标价」等问题，路由层直接拒答，不进入执行层。

LLM 不可用时，系统自动降级：直接输出 blocks 与原文，前端永不白屏。

## 【2:40 - 3:00】收尾与价值总结

总结三点：

**对 CTO**：这是一套可审计、可降级、可扩展的流水线架构。  
数据流显式可控，不依赖 LangChain 等黑盒框架，金融级数据准确性有机制保障。

**对 CEO**：合规内置、风险可控。  
投资建议一律拒答，数值来源可追溯，符合持牌信息平台的合规要求。  
结合华尔街见闻自有内容，可构建「以平台视角回答金融问题」的专属 AI，形成差异化护城河。

**对业务**：从被动问答到主动预警、从单资产到组合分析，有清晰的演进路线图。
