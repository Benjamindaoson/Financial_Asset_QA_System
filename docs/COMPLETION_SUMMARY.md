## 🎯 新增 API 端点

### 性能监控（4个）
- `GET  /api/monitoring/metrics` - 获取性能指标
- `GET  /api/monitoring/health/detailed` - 详细健康检查
- `GET  /api/monitoring/stats/endpoints` - 端点统计
- `POST /api/monitoring/stats/reset` - 重置统计

### 认证系统（6个）
- `POST   /api/auth/token` - 登录获取 Token
- `POST   /api/auth/logout` - 登出
- `POST   /api/auth/api-key` - 创建 API Key
- `DELETE /api/auth/api-key/{api_key}` - 撤销 API Key
- `GET    /api/auth/api-keys` - 列出所有 API Keys
- `GET    /api/auth/verify` - 验证认证状态

### 会话管理（9个）
- `POST   /api/sessions` - 创建会话
- `GET    /api/sessions/{session_id}` - 获取会话
- `PUT    /api/sessions/{session_id}` - 更新会话
- `DELETE /api/sessions/{session_id}` - 删除会话
- `POST   /api/sessions/{session_id}/messages` - 添加消息
- `GET    /api/sessions/{session_id}/messages` - 获取消息历史
- `GET    /api/sessions/{session_id}/context` - 获取上下文
- `GET    /api/sessions` - 列出所有会话
- `POST   /api/sessions/cleanup` - 清理过期会话
- `GET    /api/sessions/{session_id}/summary` - 获取会话摘要
