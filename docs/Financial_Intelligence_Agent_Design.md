### 6.3 风险与应对

**技术风险**:

   - 应对：实现缓存策略，添加 Alpha Vantage 备用

   - 应对：使用 Haiku 做快速分析，Sonnet 做深度分析

   - 风险：免费 API 数据可能不准
   - 应对：实现数据验证机制，多源对比

**业务风险**:

1. **合规风险**
   - 风险：可能被认定为"投资建议"
   - 应对：明确免责声明，区分"数据"和"分析"

2. **用户期望风险**
   - 风险：用户期望直接荐股
   - 应对：产品引导，强调"决策支持"而非"投资建议"
