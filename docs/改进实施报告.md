## 1. ✅ LLM Function Calling 查询路由

### 实现内容
- **新增文件**: `backend/app/routing/llm_router.py`
- **核心功能**: 使用DeepSeek模型的function calling能力自动路由查询

### 技术亮点
```python
# 定义5个路由工具
- get_stock_price: 获取实时股价和历史数据
- search_knowledge: 搜索金融知识库
- search_news: 搜索市场新闻
- compare_stocks: 对比多个股票
- get_company_info: 获取公司信息
```

### 优势
- **智能路由**: LLM自动理解用户意图，选择合适工具
- **多工具组合**: 可同时调用多个工具（如"特斯拉为什么大涨" → 价格+新闻）
- **中文支持**: 自动识别中文公司名转换为股票代码
- **降级机制**: LLM不可用时自动回退到规则路由

### 集成方式
```python
# AgentCore 新增参数
agent = AgentCore(use_llm_routing=True)  # 启用LLM路由
```

## 5. ✅ RAG Pipeline Metadata过滤

### 实现内容
- **解析YAML frontmatter**: 自动提取metadata
- **Metadata传递**: 从文档→chunk→检索结果
- **分类过滤**: 支持按category过滤检索结果

### 核心代码
```python
# 解析frontmatter
metadata, clean_content = self._parse_frontmatter(content)

# 过滤检索结果
def _filter_by_category(candidates, category):
    return [c for c in candidates
            if category.lower() in c['metadata']['category'].lower()]

# 使用过滤
results = await rag_pipeline.search(
    query="什么是市盈率",
    category_filter="估值指标"  # 只返回估值指标类文档
)
```

### 优势
- **精准检索**: 按类别过滤，减少噪音
- **灵活扩展**: 支持多种metadata字段过滤
- **性能优化**: 在检索后过滤，不影响索引性能
