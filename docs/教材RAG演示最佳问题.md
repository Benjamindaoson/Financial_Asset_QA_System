# 教材RAG演示最佳问题清单

## 🎯 设计原则

基于新注入的专业教材数据（29个文件），设计能最大化展示RAG系统亮点的问题：

### RAG系统的核心亮点
1. **专业知识准确性** - 来自官方教材，非网络碎片化信息
2. **结构化检索** - 通过metadata精准定位章节
3. **多文档对比** - 同时检索多个概念进行对比
4. **置信度评分** - 基于来源可靠性的置信度计算
5. **来源标注** - 每个答案都标注具体教材章节
