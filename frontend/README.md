# 金融资产问答系统 - 前端使用指南

## 界面概览

根据 Figma 设计稿实现的完整前端界面，包含以下功能模块：

### 1. 顶部导航栏
- **Logo 和标题**：金融资产问答系统 (BETA)
- **功能按钮**：
  - 反馈：提交用户反馈
  - 帮助：查看使用帮助
  - 清除记录：清空聊天历史

### 2. 警告横幅
- 显示免责声明：本系统所有内容仅供参考，不构成投资建议

### 3. 主聊天区域

#### 欢迎界面（首次进入）
- 欢迎图标和标题
- 系统介绍文字
- 4个快捷操作按钮：
  - 📊 查股价
  - 📈 看涨跌
  - 📚 学概念
  - 🔍 看大盘

#### 快捷问题标签
点击即可快速提问：
- 阿里巴巴当前股价是多少？
- 最近7天特斯拉涨幅？
- 市盈率是什么？
- 比特币今日行情
- A股今日行情
- 苹果公司股价

#### 聊天消息
- 用户消息：蓝色气泡，右对齐
- AI 回复：灰色气泡，左对齐
- 支持流式输出
- 显示工具调用状态
- 显示数据来源

### 4. 右侧边栏

#### 热门资产快捷入口
点击可快速查询股价：
- 苹果 (AAPL)
- 特斯拉 (TSLA)
- 阿里巴巴 (BABA)
- 英伟达 (NVDA)
- 比特币 (BTC)
- A股-上证指数

#### 实时行情 TICKER
显示实时股票价格和涨跌幅：
- AAPL - 苹果
- TSLA - 特斯拉
- BABA - 阿里巴巴
- NVDA - 英伟达
- BTC - 比特币
- AMZN - 亚马逊
- 上证 - 上证指数

涨跌幅用颜色标识：
- 🟢 绿色上箭头：上涨
- 🔴 红色下箭头：下跌

#### 常见问题 FAQ
- 如何查询股票价格？
- 什么是市盈率？
- 如何分析财务报表？
- 技术分析指标有哪些？

### 5. 输入区域
- 大输入框，支持多行输入
- 占位符提示：请输入您的金融问题，例如「阿里巴巴当前股价是多少？」
- 发送按钮（带 Enter 提示）
- 底部提示：Shift + Enter 换行

## 启动方式

### 方法 1：使用一键启动脚本（推荐）
```bash
start-all.bat
```

### 方法 2：分别启动

#### 启动后端
```bash
cd backend
venv\Scripts\activate
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

#### 启动前端
```bash
cd frontend
npm run dev
```

## 访问地址

- **前端界面**：http://localhost:3000
- **后端 API**：http://localhost:8000
- **API 文档**：http://localhost:8000/docs

## 功能特性

### 交互体验
- ✅ 流式响应：实时显示 AI 回复
- ✅ 工具调用可视化：显示正在调用的工具
- ✅ 数据卡片：股票信息以卡片形式展示
- ✅ 趋势图表：历史数据以图表形式展示
- ✅ 快捷操作：一键查询热门资产

### 响应式设计
- 适配不同屏幕尺寸
- 流畅的动画效果
- 清晰的视觉层次

### 用户体验优化
- 加载状态提示
- 错误信息展示
- 自动滚动到最新消息
- 键盘快捷键支持（Enter 发送，Shift+Enter 换行）

## 技术栈

- **React 18**：UI 框架
- **Vite**：构建工具
- **Tailwind CSS**：样式框架
- **Recharts**：图表库
- **SSE (Server-Sent Events)**：流式响应

## 注意事项

1. 确保后端服务已启动并运行在 8000 端口
2. 首次启动需要配置 API 密钥（通过 setup.py）
3. 历史记录存储在浏览器本地，刷新后会清空
4. 实时行情数据为示例数据，实际使用需要连接真实 API
