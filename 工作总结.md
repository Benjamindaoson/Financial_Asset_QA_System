## 🚀 使用方式

### 1. 自动模型选择（推荐）
系统会自动根据查询复杂度选择最合适的模型：

```python
from app.agent import AgentCore

agent = AgentCore()

# 简单查询 → 自动选择DeepSeek (便宜)
async for event in agent.run("苹果股价"):
    print(event)

# 复杂查询 → 自动选择Claude (高质量)
async for event in agent.run("分析美联储加息对科技股的影响"):
    print(event)
```

### 2. 手动指定模型
如果需要，可以强制使用特定模型：

```python
# 强制使用Claude
async for event in agent.run("苹果股价", model_name="claude-opus"):
    print(event)

# 强制使用DeepSeek
async for event in agent.run("苹果股价", model_name="deepseek-chat"):
    print(event)
```

### 3. API调用
```bash
# 自动选择模型
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"query": "苹果股价"}'

# 指定模型
curl -X POST "http://localhost:8000/api/chat?model=deepseek-chat" \
  -H "Content-Type: application/json" \
  -d '{"query": "苹果股价"}'

# 查看可用模型和使用统计
curl http://localhost:8000/api/models
```

## 🔧 如何添加新模型

只需3步即可添加新模型（如Qwen、GPT-4等）：

### 步骤1: 添加API Key到 `.env`
```env
QWEN_API_KEY=your_api_key_here
```

### 步骤2: 在 `config.py` 添加配置
```python
QWEN_API_KEY: Optional[str] = None
```

### 步骤3: 在 `multi_model.py` 注册模型
```python
if qwen_key:
    self.add_model(
        name="qwen-turbo",
        config=ModelConfig(
            provider=ModelProvider.QWEN,
            model_name="qwen-turbo",
            api_key=qwen_key,
            base_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
            cost_per_1m_input=0.3,
            cost_per_1m_output=0.6,
            priority=7
        )
    )
```
