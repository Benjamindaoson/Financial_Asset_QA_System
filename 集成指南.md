# 免费金融API集成 - 中文安装指南

## 📋 概述

本指南帮助你集成8个免费金融API到系统中：

1. **Alpha Vantage** - 股票数据，25次请求/天
2. **Finnhub** - 股票数据，60次请求/分钟
3. **FRED** - 经济指标，无限制
4. **Polygon.io** - 股票数据，5次请求/分钟
5. **Twelve Data** - 多资产数据，有限积分
6. **Financial Modeling Prep (FMP)** - 财务报表
7. **CoinGecko** - 加密货币价格，无需API密钥
8. **Frankfurter** - 外汇汇率，无需API密钥

## 🚀 快速开始

### 1. 安装依赖

```bash
cd backend
pip install -r requirements.txt
```

新增的依赖包括：
- `alpha-vantage` - Alpha Vantage官方库
- `fredapi` - FRED官方库
- `requests` - HTTP请求
- `pandas` - 数据处理
- `numpy` - 数值计算

### 2. 获取API密钥

#### Alpha Vantage（免费 - 25次请求/天）
1. 访问：https://www.alphavantage.co/support/#api-key
2. 输入邮箱
3. 立即获得API密钥

#### Finnhub（免费 - 60次请求/分钟）⭐ 推荐
1. 访问：https://finnhub.io/register
2. 邮箱注册
3. 从控制台获取API密钥

#### FRED（免费 - 无限制）⭐ 推荐
1. 访问：https://fred.stlouisfed.org/docs/api/api_key.html
2. 创建账户
3. 申请API密钥（即时批准）

#### Polygon.io（免费 - 5次请求/分钟，延迟15分钟）
1. 访问：https://polygon.io/dashboard/signup
2. 注册
3. 从控制台获取API密钥

#### Twelve Data（免费 - 有限积分）
1. 访问：https://twelvedata.com/pricing
2. 注册免费计划
3. 获取API密钥

#### Financial Modeling Prep（有免费层）
1. 访问：https://site.financialmodelingprep.com/developer/docs/
2. 注册
3. 获取API密钥

#### CoinGecko 和 Frankfurter
- 无需API密钥！✅ 直接使用

### 3. 配置环境变量

在 `.env` 文件中添加：

```bash
# 现有配置
ANTHROPIC_API_KEY=你的anthropic密钥
REDIS_HOST=localhost
REDIS_PORT=6379

# 新增金融API密钥
ALPHA_VANTAGE_API_KEY=你的alpha_vantage密钥
FINNHUB_API_KEY=你的finnhub密钥
FRED_API_KEY=你的fred密钥
POLYGON_API_KEY=你的polygon密钥
TWELVE_DATA_API_KEY=你的twelve_data密钥
FMP_API_KEY=你的fmp密钥

# CoinGecko和Frankfurter无需密钥
```

## 🎯 常见使用场景

### 1. 多资产投资组合追踪

```python
async def track_portfolio():
    service = EnhancedMarketDataService()

    portfolio = {
        "AAPL": 10,      # 10股苹果
        "GOOGL": 5,      # 5股谷歌
        "茅台": 2,        # 2股茅台
        "BTC-USD": 0.5   # 0.5个比特币
    }

    total_value = 0
    for symbol, quantity in portfolio.items():
        price_data = await service.get_price(symbol)
        value = price_data.price * quantity
        total_value += value
        print(f"{symbol}: ${value:,.2f}")

    print(f"投资组合总价值: ${total_value:,.2f}")
```

### 2. 经济数据仪表板

```python
async def economic_dashboard():
    service = EnhancedMarketDataService()

    indicators = {
        "GDP": "国内生产总值",
        "UNRATE": "失业率",
        "CPIAUCSL": "通货膨胀率(CPI)",
        "FEDFUNDS": "联邦基金利率"
    }

    for series_id, name in indicators.items():
        data = await service.get_economic_indicator(series_id)
        if data:
            latest = data['data'][max(data['data'].keys())]
            print(f"{name}: {latest}")
```

### 3. 货币转换器

```python
async def convert_currency(amount: float, from_curr: str, to_curr: str):
    service = EnhancedMarketDataService()

    rate_data = await service.get_forex_rate(from_curr, to_curr)
    rate = rate_data['rates'][to_curr]
    converted = amount * rate

    print(f"{amount} {from_curr} = {converted:.2f} {to_curr}")
    return converted

# 使用示例
await convert_currency(100, "USD", "CNY")  # 100美元转人民币
await convert_currency(1000, "CNY", "USD")  # 1000人民币转美元
```

### 4. 股票对比分析

```python
async def compare_stocks(symbols: list):
    service = EnhancedMarketDataService()

    for symbol in symbols:
        price = await service.get_price(symbol)
        change = await service.get_change(symbol, days=7)
        info = await service.get_info(symbol)

        print(f"\n{symbol} - {info.name}")
        print(f"当前价格: ${price.price}")
        print(f"7日涨跌: {change.change_pct}%")
        print(f"趋势: {change.trend}")
        print(f"市盈率: {info.pe_ratio}")

# 使用示例
await compare_stocks(["AAPL", "GOOGL", "MSFT", "茅台"])
```
